# Language Review & Sign-off

**Owner:** Language Champions (crew:lang-champions)

Quality review of keyword mappings and transpiled code samples for each target language.

## Review Process

For each language (zh, am, ur):

1. **Keyword review** — Verify Legesher keyword mappings are correct and natural
2. **Sample review** — Review 50 transpiled files for correctness and readability
3. **Model output review** — Spot-check model outputs in target language (Week 2)
4. **Sign-off** — Formal approval that transpiled code is ready for training

## Directory Structure

Each language has its own folder with review documents:

```
zh/  — Chinese review notes and sign-off
am/  — Amharic review notes and sign-off
ur/  — Urdu review notes and sign-off
```

## Sign-off Gate

Final sign-off on all 3 languages (Day 5) is a **gate for training** — training cannot begin until all languages are approved.
